Optimisation du site - Introduction

le 26 août 2015
  • gulp
  • web
  • .htaccess
  • composer
  • bower

Depuis quelques jours je travaille sur l'optimisation de mon site personnel. Au passage j'ai souhaité comparer différents outils (tels que gulp, composer avec PHP, ...) à ceux que j'utilise d'habitude (grunt, npm avec NodeJs, ...). Tant que j'y étais, je me suis dit pourquoi ne pas rédiger un petit blog expliquant la mise en place de ces outils et les différentes techniques habituelles pour optimiser un site web.

J'évoquerai notamment :
- Gulp, un outil pour créer des tâches executables en ligne de commandes
- Composer, un gestionnaire de packages (orienté PHP)
- Bower, un gestionnaire de packages (orienté front : librairies JS, frameworks CSS)
- La conf Apache qui permet d'optimiser également votre site web

Avant de commencer on va faire un bilan rapide de l'existant. J'utilise pour cela les notes Y-Slow et PageSpeed calculées sur le site gtmetrix.com. On voit ci-dessous que la note Y-Slow est B (82%) et celle de PageSpeed est C (77%).
